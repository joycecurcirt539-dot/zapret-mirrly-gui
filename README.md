**Zapret Mirrly GUI** — это современная, быстрая и эстетичная графическая оболочка (GUI) для популярной утилиты обхода блокировок DPI `zapret` (winws), созданная с использованием **WinUI 3 (Windows App SDK)** и **.NET 10**.

Приложение разработано как часть экосистемы **Mirrly** и базируется на конфигурациях утилиты обхода блокировок `zapret`. Оно позволяет управлять службами winws без необходимости работы с командной строкой и ручной правки файлов реестра или планировщика.

> [!WARNING]
> Для корректной работы приложения (установка служб, запуск winws, использование WinDivert драйвера) **требуются права администратора**. При запуске приложение автоматически запросит повышение прав UAC.

## ✨ Основные возможности

* 🖥️ **Премиальный современный UI:** Интерфейс в стиле Fluent Design с поддержкой темных и светлых тем, эффектом акрила/слюды (Mica/Acrylic) и плавными микроанимациями.
* ⚙️ **Управление Windows-службой:** Установка, удаление, запуск и остановка службы `winws` в один клик. Служба работает в фоне даже после закрытия интерфейса приложения.
* ⚡ **Менеджер пресетов:** Быстрый выбор стратегий обхода DPI (включая FAKE TLS, SIMPLE FAKE, ALT-пресеты) для YouTube, Discord и Telegram. Также есть возможность задавать собственные аргументы.
* 🔍 **Встроенная диагностика сети:** Тестирование доступности Discord (голосовые каналы и шлюзы), YouTube (видеохостинги и API) и любых пользовательских сайтов прямо из программы. DNS и Ping-тесты.
* 📝 **Просмотр логов в реальном времени:** Интегрированная консоль логов, отображающая вывод утилиты `winws.exe` в живом режиме для быстрого выявления проблем.
* 📋 **Редактор списков доменов:** Удобный интерфейс для добавления сайтов в черный (blacklist) и белый (whitelist) списки или редактирования файла пользовательских хостов.
* 📦 **Полная автономность (Self-Contained):** Приложение скомпилировано в единый файл (`ZapretMirrlyGUI.exe`), который не требует предварительно установленного .NET Runtime на компьютере.

## 📂 Архитектура и разделы

Приложение разделено на несколько удобных экранов:

1. **Панель управления (Dashboard):** Главный экран с переключателем состояния службы (Запущена/Остановлена), выбором сетевого интерфейса и выбором предустановленного пресета обхода.
2. **Диагностика (Diagnostics):** Инструмент для проверки работоспособности. Проверяет доступность Discord API, Discord Media, YouTube Video, выполняет трассировку и DNS-запросы для выявления блокировок провайдера.
3. **Списки доменов (Lists):** Позволяет управлять списками сайтов. Вы можете прямо из приложения редактировать локальные файлы конфигурации `zapret`.
4. **Журнал событий (Logs):** Показывает живой лог `winws.exe`, помогая понять, применяются ли правила обхода к проходящему трафику.
5. **Настройки (Settings):** Конфигурация автозапуска приложения, скрытия в трей при закрытии, выбор DNS-серверов (Google, Cloudflare, Yandex, системный) и тонкая настройка параметров.
6. **Поддержка (Support):** Ссылка на страницу даната автора через daLink, а также информация об экосистеме Mirrly.

## 🖥️ Системные требования

* **Операционная система:** Windows 10 (сборка 1809 и выше) или Windows 11 (x64)
* **Права:** Права Администратора (для работы драйвера WinDivert)
* **Сеть:** Активное интернет-подключение без сторонних конфликтующих DPI-обходчиков (например, GoodbyeDPI должен быть остановлен перед запуском службы).

## 🚀 Установка и запуск

1. Перейдите в раздел **[Releases](https://github.com/joycecurcirt539-dot/zapret-mirrly-gui/releases)** этого репозитория.
2. Скачайте последнюю версию исполняемого файла `ZapretMirrlyGUI.exe`.
3. Запустите файл от имени Администратора.
4. Выберите нужный пресет в панели управления (например, `general (FAKE TLS AUTO).bat`).
5. Нажмите кнопку **«Запустить»** или **«Установить службу»** (если хотите, чтобы обход работал автоматически при включении ПК).
6. Пользуйтесь свободным интернетом!

## 🛠️ Как это работает

При первом запуске приложение распаковывает необходимые ресурсы (включая оригинальный архив `zapret.zip` с утилитой `winws.exe`, драйвером `WinDivert` и конфигурационными скриптами) в локальную папку пользователя `%LOCALAPPDATA%\ZapretMirrlyGUI`.

При нажатии кнопки запуска GUI формирует команду запуска с выбранными параметрами и передает её службе Windows или запускает процесс локально. `winws` перехватывает сетевые пакеты через `WinDivert` и модифицирует заголовки пакетов (разделяет TCP-сегменты, меняет регистр букв в host-заголовках, отправляет фейковые TLS-запросы), обходя алгоритмы DPI провайдеров.
