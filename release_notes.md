Zapret Mirrly GUI v1.1.5 — Нативный C# диагностический движок, управление службами и интеграция API Яндекса

В версии 1.1.5 представлена полностью переработанная архитектура диагностического модуля, перенесенного с PowerShell на нативный C# (.NET 10), а также прямой контроль системных служб Windows и интеллектуальный парсер стратегий обхода.

Ключевые изменения и улучшения:

1. Нативный диагностический движок на C#
- Полный отказ от запуска внешних PowerShell-скриптов (`test zapret.ps1`) во время работы в GUI, что полностью исключает мигание консольных окон на экране пользователя.
- Все сетевые тесты (проверка DNS, пинг, доступность HTTP, TLS 1.2, TLS 1.3) реализованы на асинхронном C# стеке, работающем параллельно и независимо.
- Вызов утилиты `curl.exe` для точной проверки DPI-блокировок осуществляется через безопасный механизм `ProcessStartInfo` с передачей нативных аргументов.

2. Измерение пинга популярных сервисов
- Добавлено детальное измерение сетевых задержек (ping) до серверов Discord, YouTube и дата-центров Telegram (DC2 и DC4) с визуальным отображением пинга в миллисекундах.

3. Автоматический подбор и рекомендация лучшей стратегии
- По завершении диагностики нативный модуль анализирует количество успешных тестов по каждой из 20 доступных стратегий обхода Flowseal.
- Приложение автоматически вычисляет наиболее эффективную конфигурацию для вашего провайдера и рекомендует её к выбору в один клик.

4. Определение интернет-провайдера (ISP)
- Интегрировано получение информации о текущем провайдере связи через Яндекс API (определение ASN, имени провайдера и региона), что упрощает подбор и анализ проблемных блокировок.

5. Расчёт времени до окончания диагностики
- Добавлен точный таймер обратного отсчёта, показывающий приблизительное время до завершения полного цикла тестирования (около 1.5 - 2 минут).

6. Нативный C# менеджер пресетов и системных служб
- Управление процессами обхода `winws.exe` и системными службами Windows теперь осуществляется напрямую через Win32 API (`advapi32.dll`) и реестр Windows.
- Полностью удален скрипт `service.bat`, что исключает ложные срабатывания антивирусов на подозрительные сценарии командной строки.
- Безопасная нативная пропись параметров TCP Timestamps (`Tcp1323Opts = 1`) в системном реестре Windows.

Установка:
Скачайте исполняемый файл ZapretMirrlyGUI.exe и запустите его от имени Администратора.
