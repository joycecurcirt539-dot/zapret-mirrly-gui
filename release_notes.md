Zapret Mirrly GUI v1.1.6 — Полное восстановление работы + Глобальные хоткеи, Оверлей и MVVM

В версии v1.1.6 представлено полное восстановление работы ядра обхода блокировок, интеграция системных глобальных горячих клавиш через Win32 API, поверхвсехоконный акриловый оверлей уведомлений, интеллектуальный мониторинг сетевых адаптеров и полный переход архитектуры на паттерн MVVM.

Ключевые изменения и улучшения:

1. Глобальные горячие клавиши (Hotkeys)
- Добавлена поддержка системных хоткеев через Win32 API. Клавиши работают из любых игр, браузеров и полноэкранных приложений:
  • Ctrl + Alt + Z — Включить / Выключить DPI обход (winws.exe)
  • Ctrl + Alt + X — Включить / Выключить Telegram Proxy
  • Ctrl + Alt + C — Глобальное переключение всех обходов одновременно

2. Компактный Акрил-Оверлей поверх всех окон (AlwaysOnTop)
- При нажатии горячих клавиш или смене сети поверх всех окон в правом верхнем углу плавно всплывает оверлей на 3.5 секунды.
- Сверхкомпактный размер (78x44px) с эффектом жидкого стекла (Acrylic Glass).
- При смене сети всплывает аккуратный акриловый кругляш (46x46px) с анимацией ProgressRing (Переподключение...).
- Отображение статуса в реальном времени: 116 мс (зелёный), Telegram 1 мс, -- (красный прочерк).

3. Автоматический мониторинг сети (NetworkMonitorService)
- Отслеживание переключения сетевых адаптеров (Wi-Fi, Ethernet, VPN, телефон) с автоматическим переподключением активного обхода.

4. Полное восстановление ядра обхода и рефакторинг на MVVM
- Восстановлена подгрузка service.bat и оригинальный 1:1 парсер батников Flowseal.
- winws.exe запрашивает права Администратора (Verb = runas) и работает 100% скрыто без окон консоли.
- Архитектура MVVM: полное разделение слоя представления (WinUI 3 XAML Pages) и бизнес-логики (ViewModels).

5. Визуальные улучшения и микро-анимации (Beta)
- Новые эффекты плавного появления элементов, мелкие фиксы багов интерфейса и вывода логов.

5. Расчёт времени до окончания диагностики
- Добавлен точный таймер обратного отсчёта, показывающий приблизительное время до завершения полного цикла тестирования (около 1.5 - 2 минут).

6. Нативный C# менеджер пресетов и системных служб
- Управление процессами обхода `winws.exe` и системными службами Windows теперь осуществляется напрямую через Win32 API (`advapi32.dll`) и реестр Windows.
- Полностью удален скрипт `service.bat`, что исключает ложные срабатывания антивирусов на подозрительные сценарии командной строки.
- Безопасная нативная пропись параметров TCP Timestamps (`Tcp1323Opts = 1`) в системном реестре Windows.

Установка:
Скачайте исполняемый файл ZapretMirrlyGUI.exe и запустите его от имени Администратора.
