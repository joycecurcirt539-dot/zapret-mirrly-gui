## Zapret Mirrly GUI v1.0.1

Релиз стабильности и оптимизации интерфейса. 

### Изменения и исправления
- **Устранение падения скрипта диагностики**: Скрипт тестирования стратегий (`test zapret.ps1`) теперь работает в неинтерактивном режиме с подавлением всех консольных чтений ввода (`ReadKey` / `Read-Host`) при запуске из GUI. Добавлено глобальное логирование ошибок диагностики.
- **Интеграция ServiceController**: Управление статусом службы переведено с парсинга консольного вывода `sc.exe` на прямое API ServiceController. Устранены ошибки отображения статуса службы в состоянии `UNKNOWN`.
- **Новый цветовой язык логов**: Текстовые логи заменены на RichTextBlock с цветовым кодированием (5 категорий: ошибки логов — красные, предупреждения — оранжевые, успехи — зеленые, команды — синие, общая информация — серые). Добавлены красивые XAML-подсказки с цветовой схемой.
- **Защита от двойных кликов**: Добавлены асинхронные индикаторы загрузки (ProgressRing) и автоматическое отключение интерактивности элементов управления во время выполнения операций.
- **Оптимизация веса сборки**: Исключены временные и тестовые директории публикации из метаданных проекта, что уменьшило размер итогового исполняемого файла с 611 МБ до стабильных ~307 МБ.

## Zapret Mirrly GUI v1.0.0

Первый публичный релиз — GUI для утилиты обхода DPI, построенный на сборке **Flowseal** ([zapret-discord-youtube](https://github.com/Flowseal/zapret-discord-youtube)) и оригинальном движке **bolvan** ([zapret](https://github.com/bolvan/zapret)).

---

### Возможности
- Управление Windows-службой `winws` в один клик — запуск, остановка, автозапуск
- Пресеты обхода DPI: FAKE TLS, SIMPLE FAKE, ALT и другие — для YouTube и Discord
- Встроенная диагностика доступности Discord и YouTube (DNS, Ping, HTTP)
- Живой лог `winws.exe` в реальном времени
- Редактор списков доменов для тонкой настройки обхода
- Современный WinUI 3 интерфейс с тёмной темой и Fluent-анимациями

---

### Важно
- Требуются **права Администратора** (для работы драйвера WinDivert)
- Обход Telegram **не гарантируется** — для Telegram используй [tg-ws-proxy](https://github.com/Flowseal/tg-ws-proxy) от Flowseal
- Скачивай только из официального репозитория, не из зеркал

### Установка
Скачай `ZapretMirrlyGUI.exe` ниже и запусти от имени Администратора. Установка не нужна — всё в одном файле.

---

### Легенды, без которых это невозможно
- **[Flowseal](https://github.com/Flowseal)** — создатель `zapret-discord-youtube` и `tg-ws-proxy`
- **[bolvan](https://github.com/bolvan)** — создатель оригинального движка `zapret` и `winws`
