# Как внести вклад в Zapret Mirrly GUI

Спасибо за интерес к проекту! Любой вклад приветствуется.

## С чего начать

1. Убедись что проблемы/идеи нет в [Issues](https://github.com/joycecurcirt539-dot/zapret-mirrly-gui/issues)
2. Открой новый Issue с описанием бага или предложения
3. Форкни репозиторий и создай ветку от `main`

## Требования к окружению

- Windows 10/11 (x64)
- [.NET 10 SDK](https://dotnet.microsoft.com/download)
- [Visual Studio 2022+](https://visualstudio.microsoft.com/) с компонентом **Windows App SDK**

## Как запустить проект локально

```bash
git clone https://github.com/joycecurcirt539-dot/zapret-mirrly-gui.git
cd zapret-mirrly-gui
dotnet restore
dotnet run
```

## Типы вкладов

| Тип | Описание |
|---|---|
| 🐛 **Bug fix** | Исправление ошибок — всегда приветствуется |
| ✨ **Feature** | Новая функциональность — обсуди в Issue сначала |
| 📝 **Docs** | Улучшение README, комментариев, Wiki |
| 🌐 **Translation** | Перевод интерфейса на другой язык |
| 🎨 **UI** | Улучшение дизайна и UX |

## Правила оформления PR

- Один PR — одна задача
- Описание на русском или английском
- Код должен компилироваться без ошибок (`dotnet build`)
- UI-изменения — скриншот до/после

## Сообщить о баге

Используй шаблон:
```
**Версия:** v1.0.0
**ОС:** Windows 11 24H2
**Пресет:** general (FAKE TLS AUTO).bat
**Что происходит:** ...
**Что ожидалось:** ...
**Лог winws:** (вставь из вкладки Логи)
```
